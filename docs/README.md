# Docs - Chef IA Studio

Documentos de referencia e historico do projeto.

- `planoCompletoChefia.md`: plano principal de migracao e arquitetura.
- `deep-research-report.md`: pesquisa de UI, bibliotecas e deploy.
- `STATUS.md`: historico do estado anterior.
- `PROXIMOS_PASSOS.md`: roteiro antigo de proximas fases.
- `FASE3_COMPLETA.md`: notas da fase 3.
- `TROUBLESHOOTING.md`: diagnosticos e comandos de debug.
- `ROADMAP_ATUAL.md`: estado real apos as limpezas e proximas fases.
- `HANDOFF_PROXIMA_ATUALIZACAO.md`: resumo de retomada com o que mudou, proximos passos e melhorias possiveis.

O app executavel fica na raiz de `app-cardapio-ia/` e em `public/` + `src/`.
